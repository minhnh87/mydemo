# SMART · Centralized Information Hub (Mockup)

Bản mockup tĩnh (HTML thuần) cho **Trung tâm Thông tin Nội bộ** của TVS — dùng để review trước khi phát triển trên web-fe.

> ⚠️ Dữ liệu trong mockup là **dữ liệu minh hoạ (dummy data)**, không phải dữ liệu thật.

## Nội dung

| File | Mô tả |
|------|-------|
| [`index.html`](./index.html) | Trang **Hub** — kênh thông tin nội bộ: thông báo khẩn, thông điệp Ban Lãnh đạo, bảng tin (lọc theo nhóm + tìm kiếm), yêu cầu chờ duyệt, liên kết nhanh. |
| [`cms.html`](./cms.html) | Trang **CMS** — quản trị nội dung: tạo/sửa/xóa bài, ghim, yêu cầu xác nhận đọc, luồng **Chờ publish → Publish**, xem trước (preview). |

## Cách xem

- **Trang Hub:** https://minhnh87.github.io/mydemo/
- **Trang CMS:** https://minhnh87.github.io/mydemo/cms.html
- **Local:** tải repo về và mở `index.html` bằng trình duyệt (không cần cài đặt gì).

## Trạng thái

Đang ở giai đoạn mockup để lấy ý kiến team. Sau khi duyệt sẽ port sang ứng dụng web-fe (Next.js + MUI).
